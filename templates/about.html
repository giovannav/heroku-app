Sistema de Identificação para Hortas

Sobre

Este sistema foi desenvolvido baseado no ODS 2 que trata do combate à fome e agricultura sustentável. Mais informações sobre o ODS podem ser encotradas neste endereço.

O objetivo do sistema é utilizar imagens de satélite para encontrar locais próprios para a implantação de hortas urbanas. Baseando-se, principalmente, nesta dissertação que analisa as hortas urbanas na cidade de João Pessoa (PB), levanta-se evidências de que a produção urbana de alimentos, principalmente aquelas feitas próximas a locais de vulnerabilidade social, aumenta a segurança alimentar das pessoas que vivem próximas às hortas.

A tecnologia utiliza para a segmentação dos locais próprios para implantação das hortas foram redes neurais U-net a partir de datasets de domínio público. Tem-se como público alvo pessoas/entidades que cuidem do planejamento urbano e da implementação de políticas públicas de abastecimento, como o SEMA em Piracicaba.

Todo o sistema foi desenvolvido por Giovanna Valim Frederico para a disciplina de Projeto Integrado em Engenharia de Computação 1 (2022.2) e está disponível no GitHub (rede neural e aplicação) para consulta pública.
